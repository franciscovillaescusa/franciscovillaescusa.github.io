Fcodes: These are the set of libraries written in python/cython to analyze the output of N-body and hydrodynamic simulations. They can be used to compute density fields, power spectra, correlation functions, populate halos with galaxies, study the neutral hydrogen properties, make images/movies...etc. They are publicly available here and the documentation can be found here.

N-GenIC-growth: This is a modification of Volker Springel's N-GenIC code to generate the initial conditions of N-body and hydrodynamic simulations with several fluids, e.g. cold dark matter + neutrinos or baryons + cold dark matter, taking into account that the growth factors/rates of the different species are different. The code is available upon request.
